## 保存文件
遇到 host 文件保存时系统提示无权限问题，可先以管理员身份运行 CMD[点击查看方法](http://www.somode.com/jiaocheng/3177.html)，
输入 notepad 调出记事本，在记事本中打开 hosts 文件，再进行操作。具体是在以管理员身份打开的CMD中，进入存储host的文件目录，然后输入**notepad host**。
